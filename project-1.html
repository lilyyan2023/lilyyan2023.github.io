FPCC Parking Entrance Redesign

Tools Used

Iterative Design
Gnatt Chart
Arduino
Figma
Team work
Presentation

Problem

Forest Preserve of Cook County (FPCC) previously used a cable-and-lock for its parking gate and a TRAFx device for data collection, which requires manually digging out each time. My four teammates and I redesigned this FPCC Parking Entrance system.

Current Cable-and-Lock System

My Role

I was responsible for communicating with client and created hardware system prototype using Arduino and designed website interface using Figma.

Solution

Our solution contains two parts.
1. A centralized cellular system to allow users to remotely control the gate and retrieve vehicle counting data.
2. An integrated local hardware system to act as a solar-powered gate and vehicle counting data collector.
