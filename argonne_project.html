Problem

As global climate change became more severe, flood happened more frequently. If we were able to use Computer Vision to estimate the flood depth directly using real-time taken images, it would help us save more lives.

My Role

It was an independent research project and I read research paper to learn some existing algorithms. I also picked up computer vision algorithm and web scraping.

Solution

1. I used web scraping tool to collect training and testing data from different social media platforms.
2. I used facial recognition algorithm to recognize humans and segmentation algorithm to segment the picture into separate parts.
3. Then, we can calculate the flood depth by following a specific formula.
