NFT Marketplace For Decentralized Loan

Tools Used

Solidity
React
System design
Cross-team collaboration

Problem

Non-fungible Tokens are unique cryptographic tokens that exist on a blockchain and cannot be replicated. Nowadays, there are many big NFT trading platforms like OpenSea, and we would like to develop a similar marketplace for auctioning NFT to allow users to borrow money.

Marketplace Website Interface

My Role

I used system design to design specific smart contract classes including auctioning, transferring NFT ownership, and the connection with front-end. I also used Solidity to code smart contracts on auctioning functionality.

Solution

We breaked into subteams and our team developed the website of NFT marketplace and users can borrow cryptos by using NFT as mortgage. If users cannot pay back the loan after the assigned duration, the NFT ownership would be transferred to our website.

Result

We successfully developed and launched the product, attracting hundreds of users.
